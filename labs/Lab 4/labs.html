Lab Description

In this lab, I created an RSS feed to syndicate my ITWS lab assignments. The feed follows the RSS 2.0 specification and includes entries for all completed labs (Labs 1-4). Each entry contains a title, description, and link to the respective lab on my deployed Azure website.

The RSS feed also includes custom CSS styling applied through an XML stylesheet reference, which formats the feed when viewed in a browser. This demonstrates the ability to style XML documents similar to HTML.

Solution File

Key Features

  • Valid RSS 2.0 feed with proper channel and item structure
  • 4 lab entries with descriptions and links
  • CSS styling applied to XML for browser display
  • Linked from homepage and projects page
