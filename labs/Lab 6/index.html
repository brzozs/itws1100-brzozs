Lab 6: JavaScript and jQuery

Interactive web development with jQuery

Overview

In this lab, I learned JavaScript and jQuery to create interactive web elements. The lab focuses on DOM manipulation, event handling, and jQuery methods like fadeIn/Out, toggle, and event delegation.

What We Did

  • Manipulated the DOM using jQuery selectors and CSS properties
  • Created click event handlers for user interactions
  • Used jQuery animations (fadeIn, fadeOut, toggle)
  • Learned event delegation for dynamic content
  • Added and removed CSS classes dynamically
